# queue 队列

在计算机数据结构中，队列是一个抽象数据结构，它的特点是先进先出（FIFO = first in,first out）。可以想象成现实场景中的排队，比如排队做核酸。

要实现队列，需要实现两个基本操作：
- put：将一个元素添加到队列的末端
- poll：将队列的首元素弹出，并返回该元素


```
            - poll ->[1]
          [2]
          [3]
          [4]
          [5]
- put ->[6]
```

与 Stack 类似，Queue 类也是一个抽象类，它的实现可以是一个数组，一个链表，一个双端队列，或者一个双向链表。这里我们使用数组实现。
